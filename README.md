# Oracle SCM Inventory Replenishment AI Agent

A smart **AI-powered Inventory Replenishment Agent** built using Python and CrewAI that simulates real-world Oracle Fusion Cloud SCM functionality.

Designed to demonstrate how custom AI agents can automate inventory decisions, reduce stockouts, and optimize replenishment — a perfect project for Oracle SCM Functional Analysts.

---

## 🎯 Project Objective

To build an intelligent agent that analyzes inventory data and automatically generates replenishment requests (Purchase Requisition / Move Orders) — just like a real Oracle SCM analyst.

---

## ✨ Features

- **Real-time Inventory Analysis** – Checks current stock vs reorder point
- **Intelligent Reorder Calculation** – Uses demand, lead time, and safety stock logic
- **Automated Replenishment Request** – Creates simulated Purchase Requisition + Move Order
- **Multi-Agent Architecture** – Two specialized agents working together (Analyst + Execution Specialist)
- **Oracle SCM Ready** – Easily extendable to connect with Oracle Fusion REST APIs
- **Professional Logging** – Clear step-by-step reasoning and output

---

## 🛠️ Tech Stack

- **Python 3.11**
- **CrewAI** – Multi-agent orchestration framework
- **LangChain** tools
- **Pandas** – Data handling
- **Groq / OpenAI** – LLM (Large Language Model)
